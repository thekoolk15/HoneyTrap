HoneyTrap
 ---------
 No dependencies needed! Runs on Python standard library.

 Python version: 3.7+
   python --version

 Docker (optional):
   docker --version
   docker-compose --version

 Optional extras for future enhancements:

   geoip2          - Map attacker IPs to countries
   requests        - Integrate with threat intel APIs
   rich            - Pretty terminal output
   flask           - Build a monitoring dashboard

 Install optional deps:
   pip install geoip2 requests rich flask
